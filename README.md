## Features

* View plant details, including description, price, and images.
* Manage plant inventory, including adding, editing, and deleting plants.
* Fully responsive.

## Technologies Used

* Frontend: React, Redux, React Router.
* Backend: Node.js, Express, MongoDB.
* Database: MongoDB.
* Design and Styling : Bootstrap, CSS.

### Prerequisites

* Node.js installed on your machine
* MongoDB installed and running on your machine
* Stripe account for payment processing
